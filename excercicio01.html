Exercício 01

Exercício sobre tags

Tag <em>

Esta tag é utilizada para dar ênfase ao texto contido em uma página HTML.

Tag <strong>

Esta tag é utilizada para dar importância ao texto

Tag <sup> e <sub>

Estas tags são usadas para definir texto sobrescrito e subescrito.
Exemplo de <sup>: 23;
Exemplo de <sub>: H2O.

Tag <ins> e <del>

A tag <ins> é utilizada para fazer inserção de texo. Exemplo: palavra
A tag <del> é utilizada para remover texto. Exemplo: palavra 2

Tag <mark>

É utilizada para marcar o texto (funciona como um marca texto).
